# git-grundlagen



Ein lokales Repository (Ablage) ist das Git-Repository auf meinem eigenen Computer. 

Hier bearbeite ich Dateien, erstelle Commits und teste Änderungen.



Ein remote Repository (z. B. auf GitHub, GitLab, BitBucket) ist eine Kopie des Projekts auf einem Server im Internet. 

Dort kann ich meine Änderungen hochladen (pushen) und mit anderen teilen oder gemeinsam daran arbeiten.
